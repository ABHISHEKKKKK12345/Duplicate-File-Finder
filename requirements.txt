# =============================================================================
# DUPFINDER — requirements.txt
# Duplicate File Finder | Enterprise Edition
# =============================================================================
#
# This application is built exclusively on Python's standard library.
# There are NO third-party runtime dependencies.
#
# Standard library modules used (for reference):
#
#   csv            — CSV report export
#   hashlib        — SHA-256 file hashing
#   json           — JSON report export + settings persistence
#   logging        — Structured file logging with rotation
#   os             — File system operations, platform detection
#   pathlib        — Object-oriented filesystem path handling
#   platform       — OS / platform detection for fonts and system paths
#   queue          — Thread-safe inter-thread communication (IPC)
#   re             — Extension parsing via regex
#   shutil         — File move operations
#   subprocess     — Trash helpers (osascript, gio, kioclient)
#   sys            — Exit codes, Python version info
#   threading      — Background scan worker thread + Event for cancellation
#   time           — Monotonic timing for elapsed/ETA calculations
#   tkinter        — GUI framework (window, widgets, dialogs)
#   tkinter.ttk    — Themed ttk widgets (Treeview, Progressbar, etc.)
#   traceback      — Error formatting for log output
#   collections    — defaultdict for size/hash grouping
#   dataclasses    — FileEntry and DuplicateGroup models
#   datetime       — Timestamps for display, logging, and report export
#   enum           — ScanState enum
#   typing         — Type hints throughout
#   ctypes         — Windows SHFileOperationW (Recycle Bin integration)
#
# =============================================================================
#
# Optional — DEVELOPMENT ONLY (build tooling, not required to run the app)

#   mypy                    — Static type checker
